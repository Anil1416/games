# Tic Tac Toe Game

A simple Tic Tac Toe game built with HTML, CSS, and JavaScript.

## How to Play

- Click on the cells to place your X or O.
- The first player to get three in a row (horizontally, vertically, or diagonally) wins.
- If all cells are filled without a winner, it's a draw.
- Click the "Reset Game" button to start a new game.

## Files

- `index.html`: The main HTML file that structures the game.
- `style.css`: Contains the styles for the game board and cells.
- `script.js`: Implements the game logic, including player turns, win checking, and reset functionality.

## Running the Game

To run the game, open `index.html` in a web browser. No server is required as it's a client-side application.

## Troubleshooting

- Ensure all files are in the same directory.
- If the game doesn't load, check that your browser supports modern JavaScript features.
- For any issues, verify the file paths in the HTML file match the actual file names.
