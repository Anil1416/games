# Snake Game

A classic Snake game implemented using HTML, CSS, and JavaScript.

## How to Play

- Use the arrow keys to control the snake.
- Eat the red food to grow and increase your score.
- Avoid hitting the walls or yourself.

## Running the Game

1. Open `index.html` in your web browser.
2. Click the "Start Game" button to begin.
3. Use arrow keys to move the snake.

## Features

- Simple and responsive design.
- Score tracking.
- Game over screen with restart option.

## Troubleshooting

- Ensure your browser supports HTML5 Canvas.
- If the game doesn't load, check that all files (index.html, style.css, script.js) are in the same directory.
- For any issues, try refreshing the page or opening in a different browser.
